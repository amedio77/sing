음악 이론 표준 사실 자료를 검증했다. 모든 원자 사실(음자리표 배치, 계이름 매핑, 조표 순서, 12평균율 주파수)이 정확하며 수정이 필요한 오류는 없다. 아래는 완결적 표준 레퍼런스다.

---

# 음악 이론 표준 사실 자료 (Canonical Reference)

> 게임 3개 학습 주제(오선·음자리표 / 계이름 매핑 / 조표 순서)와 오디오 주파수의 단일 진실 소스.
> **표기 규약**: 옥타브 번호는 과학적 음높이 표기법(SPN), 가온다 = **C4**. 계이름은 **고정도(fixed do)** 기준.

---

## 1. 오선과 음자리표 (Staff & Clef)

### 1-1. 기본 개념
| 항목 | 내용 |
|---|---|
| 오선(staff) | 5개의 **줄(line)** + 4개의 **칸(space)** |
| 번호 세는 방향 | **아래→위** (1번 줄 = 맨 아래 줄) |
| 덧줄(ledger line) | 오선 범위를 벗어난 음을 짧은 줄로 표시 |
| 가온다(middle C) | **C4**, 피아노 한가운데 도 |
| 큰보표(grand staff) | 높은음자리표 + 낮은음자리표를 세로로 묶은 것, **C4를 공유** |

### 1-2. 높은음자리표 (Treble clef / G clef)
- **기준음: 둘째 줄 = G4.** G clef 나선의 중심이 둘째 줄을 감싸 그 줄을 "솔(G4)"로 지정 → **G 음자리표**.

| 위치 | 아래→위 순서 | 음이름(옥타브) | 니모닉 |
|---|---|---|---|
| 줄(line) 1→5 | E4, G4, B4, D5, F5 | **Every Good Boy Does Fine** |
| 칸(space) 1→4 | F4, A4, C5, E5 | **F-A-C-E** |

- 아래 첫 덧줄 = **C4 (가온다)**

### 1-3. 낮은음자리표 (Bass clef / F clef)
- **기준음: 넷째 줄 = F3.** F clef의 두 점(:)이 넷째 줄을 위아래로 감싸 그 줄을 "파(F3)"로 지정 → **F 음자리표**.

| 위치 | 아래→위 순서 | 음이름(옥타브) | 니모닉 |
|---|---|---|---|
| 줄(line) 1→5 | G2, B2, D3, F3, A3 | **Good Boys Do Fine Always** |
| 칸(space) 1→4 | A2, C3, E3, G3 | **All Cows Eat Grass** |

- 위 첫 덧줄 = **C4 (가온다)**

### 1-4. 두 음자리표 관계 요약
| 사실 | 값 |
|---|---|
| 공유 음 | 높은음자리표 아래 첫 덧줄 = 낮은음자리표 위 첫 덧줄 = **C4** |
| G clef 기준음 | 둘째 줄 = **G4** |
| F clef 기준음 | 넷째 줄 = **F3** |
| 음이름 순환 | A~G 7개 반복, 줄→칸→줄 한 단계 오를 때마다 알파벳 +1 |

## 2. 계이름 ↔ 영어 음이름 매핑 (고정도, Fixed Do)

| 계이름 | 영어 음이름 | 가온다 기준 표기 |
|---|---|---|
| 도 | C | C4 (가온다) |
| 레 | D | D4 |
| 미 | E | E4 |
| 파 | F | F4 |
| 솔 | G | G4 |
| 라 | A | A4 |
| 시 | B | B4 |
| (도) | C | C5 (한 옥타브 위) |

- **고정도**: 조(key)와 무관하게 음의 절대 이름이 고정. 도는 언제나 C.
- **옥타브 경계**: 음이름은 C에서 시작해 B에서 끝난다. 따라서 B4 다음 음 = C5 (옥타브 번호 +1).

---

## 3. 조표 붙는 순서 (Order of Accidentals)

### 3-1. 샤프(♯) 순서 — 7개: **F → C → G → D → A → E → B**

| 순서 | 1 | 2 | 3 | 4 | 5 | 6 | 7 |
|---|---|---|---|---|---|---|---|
| 영어 | F♯ | C♯ | G♯ | D♯ | A♯ | E♯ | B♯ |
| 계이름 | 파 | 도 | 솔 | 레 | 라 | 미 | 시 |

- 니모닉(영어): **F**ather **C**harles **G**oes **D**own **A**nd **E**nds **B**attle
- 니모닉(계이름): 파-도-솔-레-라-미-시

### 3-2. 플랫(♭) 순서 — 7개: **B → E → A → D → G → C → F**

| 순서 | 1 | 2 | 3 | 4 | 5 | 6 | 7 |
|---|---|---|---|---|---|---|---|
| 영어 | B♭ | E♭ | A♭ | D♭ | G♭ | C♭ | F♭ |
| 계이름 | 시 | 미 | 라 | 레 | 솔 | 도 | 파 |

- 니모닉(영어): **B**attle **E**nds **A**nd **D**own **G**oes **C**harles' **F**ather
- 니모닉(계이름): 시-미-라-레-솔-도-파

### 3-3. 핵심 관계
| 사실 | 내용 |
|---|---|
| 역순 관계 | 샤프 순서(F C G D A E B)를 뒤집으면 플랫 순서(B E A D G C F) |
| 간격 | 인접 조표는 **완전5도** 간격 (샤프는 완전5도 위, 플랫은 완전5도 아래) |

### 3-4. 조표 개수별 장조(Major key) 표

| 샤프 수 | 붙는 샤프(누적) | 장조 | | 플랫 수 | 붙는 플랫(누적) | 장조 |
|---|---|---|---|---|---|---|
| 0 | (없음) | **C장조** | | 0 | (없음) | **C장조** |
| 1 | F♯ | **G장조** | | 1 | B♭ | **F장조** |
| 2 | F♯ C♯ | **D장조** | | 2 | B♭ E♭ | **B♭장조** |
| 3 | F♯ C♯ G♯ | **A장조** | | 3 | B♭ E♭ A♭ | **E♭장조** |
| 4 | F♯ C♯ G♯ D♯ | **E장조** | | 4 | B♭ E♭ A♭ D♭ | **A♭장조** |
| 5 | +A♯ | **B장조** | | 5 | +G♭ | **D♭장조** |
| 6 | +E♯ | **F♯장조** | | 6 | +C♭ | **G♭장조** |
| 7 | +B♯ | **C♯장조** | | 7 | +F♭ | **C♭장조** |

- **빠른 판별 규칙**
  - 샤프 조: 맨 마지막 샤프의 **반음 위**가 으뜸음 (예: 마지막 F♯ → G장조)
  - 플랫 조(2개 이상): **끝에서 두 번째 플랫**이 곧 으뜸음 (예: B♭ E♭ → B♭장조). *단, 플랫 1개(B♭)는 예외적으로 F장조.*

## 4. 오디오용 주파수 (12평균율, Equal Temperament)

### 4-1. 공식
```
f = 440 × 2^((n − 69) / 12)
```
| 항목 | 값 |
|---|---|
| n | MIDI 음 번호 (A4 = 69, C4 = 60, C5 = 72) |
| 기준 | **A4 = 440 Hz** (정의값), C4 ≈ 261.63 Hz (계산값) |
| 반음(1) 상승 | × 2^(1/12) ≈ ×1.059463 |
| 옥타브(12반음) 상승 | × 2 |

### 4-2. C4~C5 한 옥타브 주파수 표 (MIDI 60~72)

| MIDI | 음이름 | 계이름 | 주파수(Hz) |
|---|---|---|---|
| 60 | C4 | 도 (가온다) | 261.63 |
| 61 | C♯4 / D♭4 | 도♯ | 277.18 |
| 62 | D4 | 레 | 293.66 |
| 63 | D♯4 / E♭4 | 레♯ | 311.13 |
| 64 | E4 | 미 | 329.63 |
| 65 | F4 | 파 | 349.23 |
| 66 | F♯4 / G♭4 | 파♯ | 369.99 |
| 67 | G4 | 솔 | 392.00 |
| 68 | G♯4 / A♭4 | 솔♯ | 415.30 |
| 69 | A4 | 라 | 440.00 (기준) |
| 70 | A♯4 / B♭4 | 라♯ | 466.16 |
| 71 | B4 | 시 | 493.88 |
| 72 | C5 | 도 (높은 도) | 523.25 |

> 소수 둘째 자리 반올림. Web Audio에서는 `oscillator.frequency.value = f`에 위 값(또는 공식 원본값)을 직접 사용.

## 부록: 핵심 원자 사실 체크리스트
- 높은음자리표 줄(아래→위): **E4 G4 B4 D5 F5** / 칸: **F4 A4 C5 E5**
- 낮은음자리표 줄(아래→위): **G2 B2 D3 F3 A3** / 칸: **A2 C3 E3 G3**
- G clef 기준음 = 둘째 줄 **G4**, F clef 기준음 = 넷째 줄 **F3**
- 가온다 = **C4**, 두 음자리표가 공유 (높은음자리표 아래 첫 덧줄 = 낮은음자리표 위 첫 덧줄)
- 고정도: 도=C 레=D 미=E 파=F 솔=G 라=A 시=B
- 샤프 순서 **F C G D A E B** / 플랫 순서 **B E A D G C F** (서로 역순)
- 주파수 공식 `f = 440 × 2^((n−69)/12)`, A4 = 440 Hz, C4 ≈ 261.63 Hz
